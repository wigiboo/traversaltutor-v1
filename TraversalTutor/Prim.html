      if weight(u, neighbor) < key[neighbor]:
        key[neighbor] = weight(u, neighbor)
        parent[neighbor] = u

ALGORITHM EXPLANATION

Prim's algorithm finds the minimum spanning tree of a connected, weighted graph. It grows the MST by always adding the cheapest edge that connects a tree vertex to a non-tree vertex, similar to Dijkstra but for spanning trees.
