DFS(graph, vertex, visited):
  visited.add(vertex)
  
  for neighbor in graph[vertex]:
    if neighbor not in visited:
      DFS(graph, neighbor, visited)

ALGORITHM EXPLANATION

DFS explores a graph as far as possible along each branch before backtracking. It uses a stack to explore deeper nodes first. Ideal for pathfinding, topological sorting, and detecting cycles.
