Topological Sort (Indegree)

PSEUDOCODE

TopologicalSortIndegree(graph):
  indegree = calculate_indegree(graph)
  queue = vertices with indegree 0
  result = []
  
  while queue is not empty:
    current = queue.dequeue()
    result.append(current)
    
    for neighbor in graph[current]:
      indegree[neighbor] -= 1
      if indegree[neighbor] == 0:
        queue.enqueue(neighbor)
  
  return result

ALGORITHM EXPLANATION

Uses Kahn's algorithm with an indegree-based approach. Starts with vertices having no incoming edges and removes them, repeating until all vertices are processed. Efficient BFS-based topological sorting.
